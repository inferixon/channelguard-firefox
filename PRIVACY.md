# Privacy Policy

ChannelGuard is a local-first browser extension.

## Data Stored Locally

ChannelGuard stores the following data in Firefox extension storage on the user's device:

- approved channel entries
- channel labels
- approval mode and expiration time
- PIN hash and salt
- local audit entries

## Data Transmission

ChannelGuard does not transmit browsing history, whitelist data, PIN data, analytics, telemetry, or personal data to the developer or any third-party server.

The extension does not use accounts, cloud sync, a remote dashboard, analytics, or tracking.

## Export And Import

Policy export/import includes whitelist policy data only. PIN hash and salt are not exported.

## Third-Party Affiliation

ChannelGuard is independent and is not affiliated with YouTube or Google.
